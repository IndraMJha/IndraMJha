## ⚡ Featured Project — Jumping Balls

> *A creative remix of Google's iconic bouncing ball doodle.*

Built entirely with vanilla JavaScript, this animation was inspired by the **Google Balls Logo** experiment.  
Each coloured ball responds to interaction — bouncing, colliding, and settling with a satisfying physics feel.

- 🔗 [**See it live →**](https://indramjha.blogspot.com/2012/05/jumping-balls.html)
- Built with: raw **JavaScript** + **HTML Canvas / DOM physics**
- Inspired by: [Google Balls Doodle](http://rawkes.com/lab/google-balls-logo)
